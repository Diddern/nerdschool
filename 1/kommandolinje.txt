Forutsetninger: *nix eller OSX eller cygwin (annet som gir linux-ish konsoll)
Ved deling av script: husk line endings i Windows kan være annerledes enn
*nix/OSX

Slides: 
    - hva forutsetter vi for denne øvelsen (se over)
    - hvordan virker skallet generelt sett (kommando --> enter --> tolkes -->
      eksekveres der og da)
    - hva er kommandoer? (små programmer) 
    - hvordan er syntax for bruk av kommandoer generelt sett? 
    - vanlige argumenter/parametre
    - pipe
    - vise script
    - henvise til fil med eksempler 

Basisverktøy: 
1) Navigering:
    - ls : hva finnes i denne katalogen?
    - pwd : hvor står jeg i filtreet?
    - cd : gå til angitt katalog
    - find : finn en fil fra angitt sted i filtreet

2) Oversikt:
    - history : vis meg kommandoene jeg har kjørt
	- tail  : les ut de siste linjene i angitt fil
	- head : les ut de første linjene i angitt fil
	- less : les meg en fil (ikke redigerbart)
	- cat : spytt ut filen til standard ut
    - grep : søk etter angitt uttrykk 

3) Fil- og katalogmanipulering:
    - mkdir : lag en katalog
    - cp/mv : kopier eller flytt
    - rm : slett
    - chmod (filrettigheter) : endre fil/katalogrettigheter
    - touch : modifiser sist-lest uten å redigere

4) Editering/Endring
    - vim : skalleditor med editerings/redigeringsmodus
    - sed : regex
    - awk : redigering av kolonner

5) Scripting basics
    - echo : skriv til skjerm (tenk System.out.println)
    - pipe | send output videre til neste kommando
    - for 
    - if
    - variabler

6) Remote
    - scp : send filer fra en maskin til en annen med ssh 
    - ssh : logg inn på remote maskin
    - rsync : synkronisering av filer (lokalt eller remote)

7) Hvordan få hjelp? 
    - google er din venn
    - man <kommando>

8) Om systemet:
    - PATH : de stiene OS-et leter etter eksekverbare filer (som python, java og
      alt annet du installerer)
    - miljøvariabler (environment variables)
    - dot-filer : skjulte filer
    - sudo : utfør som superbruker (with great power comes great responsibility)

9) Scripting: 
    - #! (shebang)
    - filrettigheter : vær obs på dette samt eierskap til filer
	- ssh (spesifikt nøkler)
	- kombinasjoner av operasjoner, som
	- find + grep (med highlight også?)
	- grep + sed + awk

Mulige oppgaver: 
	- finne fil(er) med innhold x, kopiere til ny katalog, gjøre noe med filen. Evt
	  finne fil som inneholder x og fil som inneholder y i samme katalog. Laste ned
	  og opp filer med ssh i script
	- lage build-og-deploy-script for remote deploy (ssh-nøkler, ssh-exec, scp)
	- følge med på loggfiler (finne, lytte på, kanskje gjøre noe intelligent for å
	  få ut informasjon)
	- rekursive operasjoner, feks finne alle test-klasser i et hierarki og strippe
	  ut testnavn evt funksjonsnavn

Finne oppgaver som ikke er så lett google-bare
